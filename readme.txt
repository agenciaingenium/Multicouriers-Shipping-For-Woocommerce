Shipping plugin for Chile with fixed rates by commune/region and premium API quotes from Multicouriers.

== Description ==

Multicouriers Shipping lets you configure:

* Fixed shipping rates by region or commune.
* Dynamic shipping quotes using the Multicouriers API.
* Diagnostics and support tools for operational troubleshooting.

== Installation ==

1. Upload the `multicouriers-shipping-for-woocommerce` folder to `/wp-content/plugins/`.
2. Activate the plugin from the WordPress admin panel.
3. Make sure WooCommerce is active.
4. Configure shipping methods in WooCommerce.
